# Tip Calculator (Python)

This is a simple command-line Tip Calculator built with Python.  
The program calculates how much each person should pay when splitting a bill, including a selected tip percentage.

## Features
- Accepts the total bill amount
- Allows the user to choose a tip percentage (10%, 12%, 15%, etc.)
- Splits the total bill evenly among multiple people
- Rounds the result to two decimal places for accurate currency output

## How It Works
1. The user enters the total bill amount.
2. The user selects the tip percentage.
3. The user enters the number of people splitting the bill.
4. The program calculates the total bill including tip.
5. The final amount each person needs to pay is displayed.


## Concepts Used
- Python input/output
- Variables and arithmetic operations
- Type conversion (`int`, `float`)
- Rounding numbers with `round()`
- Basic user interaction
